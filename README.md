# Back Issue Radar

Back Issue Radar is a comic collection workspace focused on helping track owned issues, spot gaps in a run, and guide future back-issue hunting.

This first scaffold centers on a compact single-screen experience for `New Mutants (2019-2022)`, with official primary cover images, quick status marking, and a structure that can grow into profile syncing, marketplace discovery, and collection intelligence.

## Current Direction

- one-screen collection portal with a darker developer-tool visual style
- full `New Mutants (2019-2022)` run loaded with primary covers
- issue-level status tracking for owned and wanted books
- groundwork for future collection sync and marketplace lookup

## Future Shape

- connect pull lists and owned collections from external sources
- identify missing issues across runs automatically
- surface buying options from eBay and other marketplaces
- build a smarter collector dashboard around runs, gaps, and cover data
